# Product

## Register

product

## Users

PC gamers who have games installed across multiple storefronts (Steam, Epic Games, manual installs) and want a unified, visually cohesive launcher to browse, launch, and track their library. They use the launcher daily or weekly to discover what to play, jump into games quickly, and see their play history at a glance.

## Product Purpose

A minimal, aesthetic desktop game launcher that detects games installed on the PC, presents them in a beautiful dark-themed library, and lets users launch them with one click. It consolidates Steam, Epic Games, and manually-installed titles into a single unified interface with cover art, metadata, play time tracking, and favorites — no ads, no storefront noise, no social features. Success means the user opens the launcher faster than they open Steam because the UI is cleaner and more pleasant.

## Brand Personality

Bold, Atmospheric, Minimalist.

The interface should feel like a premium dark space — confident typography, generous negative space, deep blacks, and subtle motion. It's a sanctuary from the noisy, sales-driven game store UIs. Every pixel earns its place.

## Anti-references

- No cluttered game store UIs (badges, discount tags, ratings, dense info on every card — that's Steam/Epic store mode, not what this is)
- No generic OS dark theme (avoid the default Windows terminal or VS Code aesthetic — it needs its own identity)
- No overly "gamer" design language (aggressive gradients, neon glows, particle effects, or angular sci-fi shapes)

## Design Principles

1. **Dark, not dim** — The dark theme should feel rich and deep (#0A0A0B), not like gray-on-gray. Contrast is high, not muddy.
2. **Content-first** — Cover art and game titles carry the interface. Chrome, badges, and UI clutter are stripped away. The game is the hero.
3. **Effortless scanning** — Users browse games in a grid; cards are compact but readable. Search and sort are immediate, not buried in menus.
4. **Deliberate motion** — Animations are few, short, and purposeful. A card lifts on hover, a panel slides in — never decorative or slow.
5. **One action, one click** — The primary action is always Play. No confirmation dialogs, no secondary prompts for the main task.

## Accessibility & Inclusion

- WCAG AA contrast minimum for body text (4.5:1)
- Respect `prefers-reduced-motion` — animations become instant transitions
- Keyboard navigable: sidebar, game grid, detail panel
- Cover images have fallback text initials when missing
